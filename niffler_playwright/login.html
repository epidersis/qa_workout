Форма авторизации для автоматического тестирования

Простая форма с валидацией полей, идеальная для отработки тестирования UI

Авторизация

Пожалуйста, введите имя пользователя
Пожалуйста, введите пароль

Данные для тестирования

Для удобства тестирования используйте следующие учетные данные:

Валидные данные:

Имя пользователя:
testuser
Пароль:
Test1234

Невалидные данные:

Имя пользователя:
wronguser
Пароль:
wrongpass

Подсказка: Форма проверяет заполнение полей и соответствие данных тестовым учетным записям.

Рекомендации для автоматического тестирования

  • Используйте id элементов для селекторов в тестах: username, password, submitBtn, resetBtn
  • Обрабатывайте ошибки валидации через элементы с id: usernameError, passwordError
  • Результат авторизации отображается в элементе с id="responseMessage"
  • Тестируйте различные сценарии: валидные данные, невалидные данные, пустые поля
  • Проверяйте функцию сброса формы с помощью кнопки "Сбросить"
  • Для тестирования можно использовать Selenium, Cypress, Puppeteer или другие инструменты
